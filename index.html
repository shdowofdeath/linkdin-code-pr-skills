Five Modes

Each skill is a slash command you run from inside any git repo in Claude Code.

📣

Project Share

/linkedin-post

Auto-analyzes your current repo and creates a ready-to-publish LinkedIn post.

  • Scans README, deps, git history
  • Strips all IP and PII automatically
  • Opens LinkedIn and composes the post
🎯

Job Match

/job-match

Compare your repo skills and resume against job descriptions to see where you stand.

  • Extracts skills from your git history
  • Strong / Partial / Missing breakdown
  • Gap analysis and recommendations
🔍

PR Share

/pr-share

Turn a specific PR into a post about the engineering lesson — not the code diff.

  • Provide a URL, branch, or "my latest PR"
  • No code diffs in the post — just insight
  • Strips ticket numbers and internals
🧠

Skills Portfolio

/skills-portfolio

Scan one or more repos and generate a ranked skills summary post.

  • Ranks by evidence strength across repos
  • Cross-references resume if provided
  • Skills in context, not just a list
📆

Weekly Digest

/weekly-digest

Auto-generate a "what I built this week" post from your git activity.

  • Groups by features, fixes, refactors
  • Highlights top 2-3 accomplishments
  • Defaults to last 7 days

Installation

Two commands. Works with any git repo in Claude Code.

# Clone the repo
git clone https://github.com/shdowofdeath/linkdin-code-pr-skills.git

# Install skills and commands
cp -R skills/* ~/.claude/skills/
cp -R commands/* ~/.claude/commands/

# Then open any repo in Claude Code and run:
/linkedin-post # or /job-match, /pr-share, /skills-portfolio, /weekly-digest
